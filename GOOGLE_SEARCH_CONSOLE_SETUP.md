# Google Search Console Setup Guide

Follow these steps to get llcuda.github.io indexed in Google search.

## Step 1: Verify Ownership

1. Go to [Google Search Console](https://search.google.com/search-console/)
2. Click "Add Property"
3. Enter: `https://llcuda.github.io/`
4. Choose verification method:
   - **HTML file upload** (recommended)
   - **HTML tag** (add to `<head>` in mkdocs.yml)
   - **Google Analytics**
   - **Domain verification**

### HTML Tag Method (Easiest)

1. Google will give you a meta tag like:
   ```html
   <meta name="google-site-verification" content="YOUR_CODE_HERE" />
   ```

2. Add to `mkdocs.yml` under `extra`:
   ```yaml
   extra:
     google_site_verification: "YOUR_CODE_HERE"
   ```

3. Rebuild and deploy:
   ```bash
   mkdocs gh-deploy
   ```

4. Click "Verify" in Search Console

## Step 2: Submit Sitemap

1. In Search Console, go to "Sitemaps" (left sidebar)
2. Enter sitemap URL: `https://llcuda.github.io/sitemap.xml`
3. Click "Submit"

MkDocs Material automatically generates:
- `sitemap.xml` - Main sitemap
- `sitemap.xml.gz` - Compressed version

## Step 3: Request Indexing for Key Pages

Manually request indexing for important pages:

1. Go to "URL Inspection" in Search Console
2. Enter each URL and click "Request Indexing":
   - `https://llcuda.github.io/`
   - `https://llcuda.github.io/guides/quickstart/`
   - `https://llcuda.github.io/guides/installation/`
   - `https://llcuda.github.io/tutorials/index/`
   - `https://llcuda.github.io/api/overview/`

## Step 4: Update robots.txt

Already created at `docs/robots.txt`:

```
User-agent: *
Allow: /

Sitemap: https://llcuda.github.io/sitemap.xml
Sitemap: https://llcuda.github.io/sitemap.xml.gz
```

## Step 5: Monitor Coverage

In Search Console:
- Check "Coverage" report for indexing status
- Fix any errors that appear
- Monitor "Enhancements" for mobile usability
